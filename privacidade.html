🔒 POLÍTICA DE PRIVACIDADE — KONTAKI

Introxeer Technology, Lda.
Última atualização: 25 de Junho de 2026

1. INTRODUÇÃO

A sua privacidade é importante para nós. Esta Política explica como recolhemos, usamos e protegemos os seus dados ao utilizar o Kontaki.

O tratamento de dados obedece à Lei n.º 7/17, de 23 de Março — Lei de Proteção de Dados Pessoais de Angola e ao RGPD (Regulamento UE 2016/679).

2. DADOS QUE RECOLHEMOS

2.1. Dados Locais. Todos os dados de negócio (produtos, vendas, clientes) são armazenados exclusivamente no seu dispositivo. Não temos acesso a eles.

2.2. Dados de Sincronização (opcional). Se ativar o backup, os dados são encriptados e transmitidos para os nossos servidores.

2.3. Dados Anónimos. Recolhemos dados estatísticos agregados (número médio de vendas, categorias populares) que nunca o identificam individualmente.

3. COMO USAMOS OS DADOS
  • Prestação do Serviço: Para que a Aplicação funcione.
  • Backup: Se ativado, para guardar os seus dados em segurança.
  • Análise Anónima: Para melhorar o produto e gerar relatórios de mercado.
  • Comunicação: Para enviar notificações sobre a Aplicação (se consentir).
4. SEGURANÇA

Os seus dados são protegidos por:

  • PIN encriptado com SHA‑256;
  • Backups protegidos com HMAC‑SHA256;
  • Comunicação com servidores via HTTPS;
  • Dados locais isolados no seu dispositivo.
5. OS SEUS DIREITOS

Nos termos dos artigos 13.º a 20.º da Lei 7/17, tem direito a:

  • Aceder aos seus dados;
  • Retificar dados incorretos;
  • Apagar os seus dados;
  • Portar os seus dados (exportar);
  • Opor‑se ao tratamento.

Para exercer estes direitos, contacte: dpo@introxeer.co.ao.
